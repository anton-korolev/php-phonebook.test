# PhoneBook.test

Тестовый проект «Телефонный справочник».

Простое веб-приложение для ведения телефонного справочника. Выводит список всех имеющихся номеров телефонов с постраничной разбивкой и позволяет добавлять, просматривать и удалять абонентов.

## Особенности реализации

- Разработано на чистом PHP 8.1, без использования фреймворков. 
- Реализует паттерн MVC.
- Список абонентов хранится в JSON.
- Для запуска приложения используется Docker Compose.

## Требования

- PHP 8.1 или новее.
- Docker Compose.

## Установка и запуск

Клонировать репозиторий:

```shell
git clone https://gitlab.com/anton-korolev/php-phonebook.test.git ~/phonebook.test
```

или

```shell
git clone https://github.com/anton-korolev/php-phonebook.test.git ~/phonebook.test
```

Установить права на запись для файла базы данных (на примере Linux):

```shell
chmod 777 ~/phonebook.test/app/db/subscriber.json
```

Запустить:

```shell
cd ~/phonebook.test
```

```shell
docker compose up -d --build phonebook.test
```

Приложение будет доступно в браузере по адресу localhost:8080 (или по IP адресу сервера на порту 8080).

## Доработка и тестирование

Исходный код проекта монтируется к Docker контейнеру. В результате все изменения в коде доступны в реальном времени, без перезапуска контейнера.

Контейнер включает настроенный Xdebug (client_port=9003).
